Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attributes give additional information about an element and are only defined in an opening element.
  • A semantic element is an element that describes its content and its role on a webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A CSS style sheet allows you to keep all you 'CSS rules' in a separate file (making design changes easier). And you do this by using a link element placed in the HTML's head element.
  • A class attribute is a name/ label given to a HTML element (in that element's opening tag in the HTML document)
  • A class selector is the 'selector' used in a CSS rule in order to apply style changes to all HTML elements with said class attribute.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git is a version control software that helps you manage and track changes to your code
  • In order to access git, use the 'git' command line.
  • git comands are not the same as bash command(ie. cd, mkdir)
